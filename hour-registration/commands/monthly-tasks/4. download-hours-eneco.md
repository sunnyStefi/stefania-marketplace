---
name: 4. download-hours-eneco
description: "Download the previous month's timesheets from Eneco Fieldglass and save them locally."
---

Use the Playwright MCP server to navigate to https://www.eu.fieldglass.cloud.sap/time_sheet_list.do?cf=1.

Read credentials from the macOS Keychain:

- username: `security find-generic-password -s worksheet-fieldglass -a username -w`
- password: `security find-generic-password -s worksheet-fieldglass -a password -w`

"Previous month" is computed in the **Europe/Amsterdam** timezone.

Download all timesheets for the previous month.

Save the files to `~/Downloads/{previous_month-previous_year}` in `MM-YY` format (e.g. `~/Downloads/04-26` for April 2026). Create the directory if it does not exist.

## Error handling

- **Login failure**: stop and report — do not retry credentials automatically.
- **No timesheets found**: stop and report — do not create an empty directory.
- **Download timeout**: wait 30 seconds and retry up to **2 times**. If still failing, report which timesheet(s) could not be downloaded and leave the directory with any files that were successfully saved.
