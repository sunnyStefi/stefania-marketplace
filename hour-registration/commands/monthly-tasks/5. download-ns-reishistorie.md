---
name: 5. download-ns-reishistorie
description: "Download the previous month's travel costs from ns.nl and save them locally."
---

Read the NS card ID from the **NS card** section of `rules/context.md`.

Use the Playwright MCP server to navigate to `https://www.ns.nl/mijnns#/reishistorie?card={NS_CARD_ID}&period=previous-month`

Read credentials from the macOS Keychain:

- username: `security find-generic-password -s worksheet-ns -a username -w`
- password: `security find-generic-password -s worksheet-ns -a password -w`

"Previous month" is computed in the **Europe/Amsterdam** timezone.

1. Select all journeys for the previous month (exclude **Automatisch saldo opladen**).
2. Select the **Download overzicht x declaraties** radio button (not **Alle transacties**).
3. Click **Download**.
4. Move the downloaded file to `~/Downloads/{previous_month-previous_year}` in `MM-YY` format (e.g. `~/Downloads/04-26` for April 2026). Create the directory if it does not exist.

## Error handling

- **Login failure**: stop and report — do not retry credentials automatically.
- **No journeys found**: stop and report before downloading — do not create an empty or misleading file.
- **Download timeout**: wait 30 seconds and retry up to **2 times** before reporting failure.
