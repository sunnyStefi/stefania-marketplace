---
name: 1. list-weekly-hours-calendar
description: "List worked hours for the current week from Google Calendar."
---

Use the Google Calendar MCP server to access my "Work" calendar and retrieve all workdays for the current week (Monday through Friday).

For each day, determine the total hours and location using the **worksheet location rules** (calendar match column).

Save the result to `~/.claude/data/worked_hours.json`. The top-level object includes an ISO week identifier so subsequent steps can verify they are operating on the current week. Dates are ISO `YYYY-MM-DD`.

```json
{
  "week": "2026-W20",
  "days": [
    { "date": "2026-05-18", "hours": 8, "location": "Eneco" },
    { "date": "2026-05-19", "hours": 8, "location": "Eneco" },
    { "date": "2026-05-20", "hours": 0, "location": "National Holiday" },
    { "date": "2026-05-21", "hours": 8, "location": "Amsterdam" },
    { "date": "2026-05-22", "hours": 8, "location": "Utrecht" }
  ]
}
```

After saving, display the contents in the console for verification.
