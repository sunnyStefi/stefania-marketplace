---
name: 4. download-hours-eneco
description: "Download the previous month's timesheets from Eneco Fieldglass and save them locally."
---

Use the Playwright MCP server to navigate to https://www.eu.fieldglass.cloud.sap/time_sheet_list.do?cf=1.

Read credentials from the macOS Keychain:

- username: `security find-generic-password -s worksheet-fieldglass -a username -w`
- password: `security find-generic-password -s worksheet-fieldglass -a password -w`

Download all timesheets for the previous month.

Save the files to `Downloads/{previous_month-previous_year}` in `MM-YY` format (e.g. `Downloads/04-26` for April 2026).
