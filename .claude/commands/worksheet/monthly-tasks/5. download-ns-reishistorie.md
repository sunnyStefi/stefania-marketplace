---
name: 5. download-ns-reishistorie
description: "Download the previous month's travel costs from ns.nl and save them locally."
---

Use the Playwright MCP server to navigate to https://www.ns.nl/mijnns#/reishistorie?card=8be53ddd-6bb2-40e0-bb33-327d98ec14fd&period=previous-month

Read credentials from the macOS Keychain:

- username: `security find-generic-password -s worksheet-ns -a username -w`
- password: `security find-generic-password -s worksheet-ns -a password -w`

1. Select all journeys for the previous month (exclude **Automatisch saldo opladen**).
2. Select the **Download overzicht x declaraties** radio button (not **Alle transacties**).
3. Click **Download**.
4. Move the files to `Downloads/{previous_month-previous_year}` in `MM-YY` format (e.g. `Downloads/04-26` for April 2026).
