# Stefania's Claude Code Marketplace

A personal marketplace of [Claude Code](https://claude.ai/code) plugins that automate consultant workflows.

## Plugins

| Plugin | Description |
|--------|-------------|
| [hours-registration](./hours-registration/README.md) | Automates the complete billing cycle for consultant invoicing |

## Installing a plugin

### From this marketplace (once hosted)

```
/plugin install hours-registration@stefania
```

### From a local clone

```bash
git clone <this-repo> ~/stefania-marketplace
```

Then in Claude Code:

```
/plugin install ~/stefania-marketplace/hours-registration
```
