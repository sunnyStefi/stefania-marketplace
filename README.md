## Installation

### From this marketplace *(once hosted)*

```sh
/plugin install hour-registration@stefania
```

### From a local clone

```sh
git clone <this-repo> ~/stefania-marketplace
```

Then in Claude Code:

```sh
/plugin install ~/stefania-marketplace/hour-registration
```
